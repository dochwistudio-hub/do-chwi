# CLAUDE.md

Do-chwi — a minimal English blog about brewing Korean *sool* (makgeolli, takju, etc.) at home. Built with Astro 5 (static output). See `STATUS.md` for current state and remaining work.

## 밥알이 이모지 작업 규칙
- 밥알이(Bap-ari) 이모지 관련 작업 시, 반드시 bapari/BAPARI_EMOJI_GUIDE.md 를 먼저 읽고 그 규칙(캐릭터 고정 설정, 생성 프롬프트 고정 문구, 명명 규칙, 팔레트)을 따를 것.
- 새 이모지가 생성/추가될 때마다 BAPARI_EMOJI_GUIDE.md 하단의 "이모지 카탈로그 > 추가 생성분" 표에 파일명·표정/동작·프롬프트 요약·용도·생성일을 반드시 기록할 것.
- 블로그 글에 이모지 삽입 시 투명 배경 버전(_t)을 쓰고, 인라인 크기는 CSS로 조절할 것.
